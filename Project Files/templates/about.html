Project Overview

OptiCrop is a Smart Agricultural Production Optimization Engine that helps farmers and agronomists choose the most suitable crop for a given plot of land. By combining soil nutrient levels with climate conditions, the system applies a trained machine learning classifier to recommend the crop most likely to thrive under those exact conditions.

Objectives

  • Analyze soil and environmental parameters to guide crop selection.
  • Compare multiple machine learning algorithms and select the best performer.
  • Deliver predictions through a simple, accessible web interface.
  • Support better agricultural planning and improved yield outcomes.

Technologies Used

Python
Flask
Scikit-learn
Pandas
NumPy
Bootstrap

Machine Learning Workflow

  1. Clean and preprocess the crop recommendation dataset.
  2. Split data into training and testing sets, then scale features.
  3. Train Logistic Regression, KNN, Decision Tree, and Random Forest models.
  4. Compare model accuracies and automatically select the best model.
  5. Persist the model and scaler using joblib for real-time prediction.

Future Enhancements

  • Live weather API integration for automatic climate data.
  • Fertilizer recommendation and yield estimation modules.
  • Regional language support for wider farmer accessibility.
  • Cloud deployment with a mobile companion application.
